# Employee Churn Prediction

## Overview

This project focuses on predicting employee churn using machine learning classification algorithms. The dataset contains employee-related information such as satisfaction level, evaluation score, number of projects, monthly working hours, department, salary, and promotion status.

## Objectives

* Perform data preprocessing and cleaning.
* Handle missing values.
* Encode categorical features.
* Train and evaluate multiple classification models.
* Compare model performance using evaluation metrics.

## Dataset Features

* Satisfaction Level
* Evaluation Score
* Number of Projects
* Average Monthly Hours
* Time Spent in Company
* Work Accident
* Promotion in Last 5 Years
* Department
* Salary
* Churn (Target Variable)

## Data Preprocessing

* Missing value treatment using median and mode.
* Label Encoding for categorical variables.
* One-Hot Encoding for salary categories.
* Feature Scaling using MinMaxScaler.
* Outlier visualization using boxplots.

## Machine Learning Models

* Logistic Regression
* K-Nearest Neighbors (KNN)
* Support Vector Machine (Linear, Polynomial, and RBF Kernels)
* Decision Tree Classifier
* Random Forest Classifier

## Evaluation Metrics

* Accuracy Score
* Precision Score
* Recall Score
* F1 Score
* Confusion Matrix
