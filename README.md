# Machine Learning Projects Portfolio

This repository contains hands-on machine learning and data analytics projects completed as part of my learning journey. The projects cover data preprocessing, exploratory data analysis, regression, classification, feature engineering, and model evaluation.

## Projects

### 1. Beer Servings Analysis

Exploratory Data Analysis (EDA) project on global beer consumption data.

**Skills Used**

* Data Cleaning
* Data Visualization
* Exploratory Data Analysis
* Pandas
* Matplotlib
* Seaborn

### 2. Restaurant Tip Prediction

Regression project for predicting restaurant tip amounts based on customer and dining information.

**Skills Used**

* Data Preprocessing
* Feature Engineering
* Regression Modeling
* Model Evaluation

### 3. Employee Churn Prediction

Classification project to predict whether an employee is likely to leave a company.

**Algorithms Used**

* Logistic Regression
* K-Nearest Neighbors (KNN)
* Support Vector Machine (SVM)
* Decision Tree
* Random Forest

**Skills Used**

* Missing Value Handling
* Feature Encoding
* Feature Scaling
* Classification Modeling
* Performance Evaluation

### 4. Social Network Ads Classification

Classification project that predicts whether a user will purchase a product based on demographic data.

**Algorithms Used**

* Logistic Regression
* K-Nearest Neighbors (KNN)
* Support Vector Machine (SVM)
* Decision Tree
* Random Forest

**Skills Used**

* One-Hot Encoding
* Feature Scaling
* Classification Modeling
* Model Evaluation
### 5. Mall Customer Segmentation

Unsupervised Machine Learning project that groups customers into distinct segments based on demographic information and spending behavior.

**Algorithms Used**

* K-Means Clustering

**Skills Used**

* Data Exploration
* Outlier Detection
* Label Encoding
* Elbow Method
* Customer Segmentation
* Cluster Visualization
* Unsupervised Learning
