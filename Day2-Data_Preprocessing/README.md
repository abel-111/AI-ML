# Data Preprocessing Project

## Objective
Perform preprocessing on the Beer Servings dataset.

## Tasks Performed

- Data loading
- Data inspection
- Histogram visualization
- Missing value handling
- Log transformation
- Outlier detection using IQR
- Label Encoding
- One-Hot Encoding

## Libraries Used

- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
