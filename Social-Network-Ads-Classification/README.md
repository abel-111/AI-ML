# Social Network Ads Classification

## Overview

This project predicts whether a user will purchase a product based on demographic information and salary data. Multiple machine learning classification algorithms are implemented and compared.

## Objectives

* Analyze user data.
* Perform preprocessing and feature engineering.
* Apply classification algorithms.
* Evaluate and compare model performance.

## Dataset Features

* Gender
* Age
* Estimated Salary
* Purchased (Target Variable)

## Data Preprocessing

* Removed User ID column.
* One-Hot Encoding for Gender.
* Feature Scaling using MinMaxScaler.
* Outlier analysis using boxplots.
* Train-Test Split for model evaluation.

## Machine Learning Models

* Logistic Regression
* K-Nearest Neighbors (KNN)
* Support Vector Machine (Linear, Polynomial, and RBF Kernels)
* Decision Tree Classifier
* Random Forest Classifier

## Evaluation Metrics

* Accuracy Score
* Precision Score
* Recall Score
* F1 Score
* Confusion Matrix

## Libraries Used

* NumPy
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn

## Learning Outcomes

* Data preprocessing
* One-Hot Encoding
* Feature scaling
* Classification techniques
* Hyperparameter selection for KNN
* Model evaluation and comparison
